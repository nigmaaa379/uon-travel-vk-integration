# Режим минимизации персональных данных

> Технический чек-лист, не юридическое заключение. Отсутствие штрафов нельзя гарантировать только программным кодом.

## Решение по ТЗ
- Разделы 1–3, Telegram и MAX сохранены.
- Указанный заказчиком мессенджер для уведомления руководителя в Разделе 1 — ВКонтакте.
- Resend удалён; email работает через российский SMTP.
- Руководитель получает в VK и email только номер обращения U-ON, без анкеты туриста.
- Для VK-уведомления используется `NOTIFY_VK_PEER_ID`; руководитель должен предварительно разрешить сообщения от сообщества.
- Завершённая VK-сессия очищается от анкеты после создания обращения в U-ON.
- `PUBLIC_SITE_ENABLED=false` до заполнения реквизитов и юридической проверки.
