# Ручное обновление акцентов продаж

Карточки слева от блока поиска тура хранятся в `web/data/sales-accents.json`.

Можно показывать от одной до трёх карточек. Для каждой карточки используются поля:

- `label` — небольшая подпись, например «Тур дня»;
- `title` — название предложения;
- `text` — короткое описание;
- `button` — текст кнопки, открывающей форму подбора.

После изменения JSON проверьте запятые и кавычки, затем пересоберите контейнер. Не размещайте неподтверждённые цены, сроки и обещания. Если у предложения есть срок действия, укажите его прямо в тексте карточки.

Пример:

```json
{
  "label": "Тур дня",
  "title": "Мальдивы для семьи",
  "text": "Актуально до 31 августа. Стоимость уточняется при бронировании.",
  "button": "Узнать варианты"
}
```

Для собственного шрифта названия в шапке предпочтителен файл `WOFF2`; запасной вариант — `WOFF`. Перед публикацией нужно проверить лицензию шрифта.
