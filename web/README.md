# Сайт «Сбежим на море»

Статический mobile-first сайт встроен в основной Node.js-сервис. Откройте `/` после запуска приложения.

## Перед публикацией
1. Замените `example.ru`, телефон, email, ссылки Telegram/WhatsApp и значения `[ИНН]`, `[ОГРН]`, `[RTA]`, адрес.
2. Вставьте официальный код Tourvisor в `#tourvisor-widget`.
3. Вставьте iframe Яндекс Бизнеса в `.yandex-widget`.
4. Добавьте идентификатор Яндекс Метрики и загружайте счётчик только после события `analytics-consent-granted`.
5. Передайте юридические тексты юристу. Текущие страницы — структурированные шаблоны, не юридическое заключение.
6. Замените OG URL и OG image, обновите `robots.txt` и `sitemap.xml`.

## Tailwind
`npm install && npm run build:css` внутри `web/`. Основной `assets/styles.css` содержит автономную тему; Tailwind-конфигурация предназначена для дальнейшей сборки и расширения.
